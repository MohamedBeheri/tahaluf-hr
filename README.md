# نظام تقييم الأداء — التحالف الوطني

تطبيق ويب لإدارة شؤون الموظفين وتقييم الأداء، ديناميكي بالكامل مع تسجيل دخول وصلاحيات على ٣ مستويات.

## المميزات
- **٣ أدوار**: مدير عام (كل شيء) · مدير إدارة (موظفو إدارته) · موظف (بياناته فقط).
- إضافة/تعديل/حذف للموظفين والإدارات والأقسام وبنود التقييم والمستخدمين.
- تقييم أداء بمعايير وأوزان، تقارير قابلة للطباعة، واستيراد/تصدير Excel.
- قاعدة بيانات سحابية **Firebase (Firestore)** + **Firebase Authentication**.
- **وضع معاينة محلي** يعمل فورًا بدون إعداد (بيانات تجريبية في المتصفح).

## التشغيل السريع (معاينة)
افتح `index.html` في المتصفح وجرّب:
`admin / 123456` (مدير عام) · `ahmed / 123456` (مدير إدارة) · `khaled / 123456` (موظف)

## النسخة الحقيقية + الرفع
راجع **[دليل_التشغيل.md](دليل_التشغيل.md)** لخطوات إعداد Firebase والنشر على Netlify.

## بنية الملفات
| الملف | الوظيفة |
|------|---------|
| `index.html` | التطبيق كاملًا |
| `config.js` | إعدادات Firebase |
| `firebase/firestore.rules` | قواعد الحماية/الصلاحيات |
| `netlify.toml` | إعداد النشر على Netlify |
